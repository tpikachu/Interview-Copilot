# 0.6.0 — 2026-06-23

Automatic updates, session resume, and a fix that brings live transcription back.

## Added

- **Automatic updates.** BrainCue now updates itself in the background — when a new
  version is published it downloads quietly and shows a one-click **"Restart to
  update"**. You can also check manually in **Settings → Software updates**.
- **Resume your last session.** Live Session now offers **"Resume last session"**
  for a profile + interview, continuing the existing session (and reloading its
  transcript) instead of creating a new one every time. Repeated rounds no longer
  pile up in Reports, and answers can build on the previous round. "New session"
  is still there when you want a fresh one.

## Changed

- **Friendlier live transcript.** It now scrolls within its own panel and sticks
  to the newest line, but pauses when you scroll up to read back — with a
  **"↓ Jump to latest"** button to re-pin it. Very long interviews are capped
  on-screen for performance (the full transcript is always in Reports), and the
  **Pause / Stop** controls stay pinned in view while you scroll.

## Fixed

- **Live transcription works again.** OpenAI retired the Realtime *Beta* API, which
  silently broke transcription (and therefore live answers). Migrated to the GA
  Realtime API. *If you were on 0.5.0, this is the fix.*
