# 0.3.0 — 2026-06-23

A refreshed dashboard chrome, live status at a glance, and first-class reset/exit
controls.

## Added

- **Custom window titlebar.** The dashboard now has its own slim titlebar that
  matches the app instead of the default OS chrome — app mark on the left,
  custom minimize / maximize-restore / close controls on the right, and a
  draggable region. Native window resizing and snapping are preserved.
- **Sidebar status panel.** The bottom of the sidebar now shows live status at a
  glance: a "Live session active / Idle" indicator plus profile and session
  counts, updating as things change. The version + "What's new" link stays here.
- **Reset App Settings.** Settings → Danger zone → *Reset* restores models,
  overlay, privacy, and keyboard shortcuts to factory defaults. Your API key and
  data are kept. Asks for confirmation first.
- **Remove all user data.** Settings → Danger zone → *Delete all* clears the
  OpenAI API key and deletes every profile and interview session (and their
  documents, notes, jobs, transcripts, answers, and reports). Asks for
  confirmation first.
- **Exit hotkey.** New global shortcut **Ctrl+Shift+Q** fully quits the app and
  stops all background processes at once. Re-bindable in Settings → Keyboard
  Shortcuts.

## Changed

- The static disclaimer text at the bottom of the sidebar was replaced by the new
  live status panel.

## Notes

- Both destructive actions are gated by a native confirmation dialog in the main
  process, so they can't run without your explicit consent.
- The exit hotkey runs the same clean shutdown as the tray "Exit", so no Electron
  helper processes are left running.
