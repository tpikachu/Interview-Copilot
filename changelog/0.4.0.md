# 0.4.0 — 2026-06-23

The app is now **BrainCue Copilot**, with a clearer floating panel and safer
Privacy Mode.

## Changed

- **Renamed to BrainCue Copilot.** New name across the window, taskbar, tray,
  installer, and docs. (Your existing local data is preserved.)
- **"Overlay" is now the "Cue Card."** The floating, always-on-top answer panel
  has a clearer name everywhere — shortcuts, tray, the tour, and in-app text.
- **The Cue Card is shown by default** on launch, and the redundant "Show / Hide
  overlay" button was removed from the sidebar. Toggle it with **Ctrl+Shift+Space**
  or from the tray.
- **Tray menu** items are clearer and now show their shortcuts: *Show BrainCue
  Dashboard*, *Show BrainCue Cue Card* (Ctrl+Shift+Space), *Privacy Mode*
  (Ctrl+Shift+H), *Exit BrainCue Copilot* (Ctrl+Shift+Q).
- **Wider content area** — dashboard pages use more of the window width (less
  empty horizontal margin).

## Added

- **Privacy Mode is protected.** It stays on by default (recommended), and
  turning it **off** now asks for confirmation first — from the tray, Settings,
  or the global shortcut — so you can't accidentally reveal the app on a screen
  share. The Settings card shows the live, configured shortcut.

## Notes

- The confirmation gate lives in the main process, so every path that could
  disable Privacy Mode goes through it.
