# 0.7.0 — 2026-06-23

Interview rounds, a searchable jobs table, client notes, richer Reports, and a
live session that follows you around the app.

## Added

- **Interview rounds, organized by type.** Each job now keeps its rounds separate
  by interview type — your **behavioral** and **technical** rounds for the same
  client no longer collapse into one. Resume the right round and answers build on
  that round's history; start a fresh one whenever you want.
- **Searchable jobs table.** Live Session lists your interviews in a paginated,
  searchable table (most recent first) — pick one to set up the round. It loads a
  page at a time, so it stays fast whether you have five jobs or five thousand.
- **New interview & Detail modals.** Adding an interview is now a focused **New
  interview** dialog; the new job appears in the table, ready to select. Each row
  has a **Detail** button to edit the job, its JD, company research, and notes —
  no more guessing whether you're editing the current one or creating a new one.
- **Client notes.** Save free-form notes per client (recruiter, panel, comp,
  quirks). They're on hand while you pick the client and inside the Cue Card
  during the interview.
- **Version at a glance.** The sidebar now shows the current version — click it to
  jump straight to **What's New**.

## Changed

- **Reports that scale.** Sessions are grouped into collapsible per-client cards
  with time analytics (how long each round ran, totals per client) — tidy even
  with thousands of clients.
- **The live session follows you.** Starting a session no longer ties it to one
  page: navigate around the app and the session — including the microphone — keeps
  running, with controls available when you return to Live Session.
- **A livelier brand.** The sidebar logo gets a soft animated glow and gradient
  wordmark (respecting your reduced-motion setting).

## Fixed

- **Cue Card buttons are clickable again.** The overlay used a single click-through
  region, so the mouse passed through the buttons. It now toggles click-through
  per region, so buttons stay clickable while the rest stays out of the way.
- **No more duplicate sessions.** Stopping a round then landing back on Live
  Session could start a brand-new session instead of offering to resume. The setup
  view now refreshes the moment a round ends, so you're correctly offered
  **Resume**.
