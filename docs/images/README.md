## Capturing

1. Load sample data (Profiles → **Load sample data**) so the UI isn't empty.
2. **Turn Privacy Mode OFF** (Settings, or `Ctrl+Shift+H`) — it excludes every window
   from screen capture, so screenshots come out blank otherwise. Turn it back on after.
3. Capture each window (Win+Shift+S / Cmd+Shift+4), crop tight, save here as PNG.
4. Aim for ~1000px wide; a dark desktop background keeps the focus on the app.
