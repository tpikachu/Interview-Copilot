## v0.3.0 (this batch)

1. **Sidebar status panel** — replaced the static disclaimer/version text at the
   bottom of the dashboard sidebar with a live status block: profile count,
   session count, and a "Live now" indicator when a session is active. Kept the
   `v<version> · What's new` changelog link.
2. **Custom titlebar** — dashboard window switched to `titleBarStyle: 'hidden'`
   (keeps native resize borders, hides the OS caption) with a custom React
   titlebar: drag region + min / maximize-restore / close controls styled to the
   app. Close hides to tray (existing behavior). New `window:*` IPC + a
   `window:maximized` push event so the maximize/restore icon stays correct.
3. **Reset features** (Settings → Danger zone), both gated by a native confirm:
   - **Reset App Settings** — clears all settings (models, overlay, privacy,
     shortcuts, consent, tour) back to factory defaults; keeps the API key + data.
   - **Remove all user data** — clears the OpenAI API key + deletes all profiles
     and sessions (FK cascade removes documents/notes/jobs/chunks/embeddings/
     transcripts/answers/reports).
   - **Exit hotkey** — new global shortcut `app:quit` (default `Ctrl+Shift+Q`)
     that fully terminates the app + all child processes at once.

### Notes / decisions

- Destructive resets confirm via a native `dialog.showMessageBox` in the main
  process, so they're gated even if the IPC is called directly.
- `app:quit` handler calls `quitApp()` (→ `before-quit` → `performShutdown`), the
  same clean teardown as the tray Exit, so no orphaned processes.
- Used `titleBarStyle: 'hidden'` rather than `frame: false` to preserve native
  window resizing/snapping on Windows.
